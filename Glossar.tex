	name=latex,
	description={LaTeX (short for Lamport TeX) is a document preparation system. The user has to think about only the content to put in the document and the software will take care of the formatting. }

{
	name=GPIO,
	description={Als GPIO (Kurzform von General-Purpose Input-Output) bezeichnet man die Pins auf einem Mikrocontroller. Diese können elektrische Signale senden und empfangen, sind aber nicht für einen spezifischen Gebrauch entwickelt}
}
